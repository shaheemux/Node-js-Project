Routes HTTP methods Description
/ GET Render Home Page index.html
Users Endpoint
/users GET Retrieving all/multiple users
/users/:id GET Retrieving a single user
/register POST Adding users and products.
/user/login GET Allowing user to login .
/users/update PATCH Updating a single user
/user/delete DELETE Deleting a single user
Products Endpoints
/products GET Retrieving multiple/all Products
/products/:id GET Retrieving a Product
/register POST Adding products.
/products/login GET Allowing product to login .
/products/update PATCH Updating a single product
/products/delete DELETE Deleting a single product
